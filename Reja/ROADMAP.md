# ADU Project Roadmap

Ushbu hujjat loyihani rivojlantirish bosqichlarini belgilaydi.

## 1-bosqich: Professional Funksiyalar
- [x] **Real-time Preview:** Har bir o'zgarishni darhol ko'rish.
- [x] **Local Storage:** Ma'lumotlarni brauzerda saqlash.
- [x] **Muhr yuklash:** Universitet muhrini rasm sifatida qo'shish.
- [x] **QR-kod generatsiyasi:** Sertifikat haqiqiyligini tekshirish uchun QR-kod qo'shish.
- [x] **Imzo yuklash:** Shaxsiy imzoni (transparent PNG) joylashtirish.
- [x] **Auto-scaling:** Uzun ismlarni avtomatik sig'dirish.

## 2-bosqich: Dizayn va Moslashuvchanlik
- [x] **Multi-Templates:** Kamida 3 xil dizayn.
- [x] **Color Picker:** Asosiy ranglarni foydalanuvchi o'zi tanlashi.
- [x] **Multi-language:** UZ, RU, EN tillarini qo'llab-quvvatlash.

## 3-bosqich: Ma'lumotlar bilan ishlash
- [x] **Batch Generation:** Excel/CSV orqali ommaviy sertifikat yaratish.
- [ ] **IndexedDB History:** Yaratilgan sertifikatlar tarixini saqlash.
- [ ] **Verification System:** QR-kod orqali onlayn tekshirish.
