# ADU Certificate Generator

Andijon Davlat Universiteti uchun maxsus sertifikat, maqtov yorlig'i va tashakkurnomalarni onlayn generatsiya qilish tizimi.

## Xususiyatlari
- **3 xil shablon:** Maqtov yorlig'i, Tashakkurnoma va Konferensiya sertifikati.
- **Jonli tahrirlash:** Ma'lumotlarni kiritish bilan bir vaqtda natijani ko'rish.
- **PDF eksport:** Tayyor hujjatni PDF formatida saqlash va chop etish (A4 Landscape).
- **Responsive Dizayn:** Har qanday qurilmada foydalanish imkoniyati.

## Texnologiyalar
- HTML5 / CSS3 (Cinzel, Montserrat, Playfair Display fontlari)
- Native JavaScript (No frameworks)

## Foydalanish
1. `index.html` faylini brauzerda oching.
2. Kerakli hujjat turini tanlang.
3. Ma'lumotlarni (F.I.Sh, sana, sabab) kiriting.
4. "PDF ga saqlash" tugmasini bosing va chop eting.

## Kelajakdagi rejalar
- [ ] O'z muhrini yuklash imkoniyati.
- [ ] QR-kod orqali sertifikatni tekshirish tizimi.
- [ ] Bir vaqtda bir nechta talaba uchun ma'lumotlarni yuklash (CSV/Excel).
